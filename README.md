R package for accessing the ArcGIS API: https://developers.arcgis.com/rest/.

## Installation

Install from GitHub using `devtools`.


```r
if(!require(devtools)) {
  install.packages("devtools")
}
devtools::install_github("NateByers/arcgisapi")

library(arcgisapi)
```
